DDNet Web

This is an experimental version of the DDNet client running in a web browser.

It currently has the following limitations:

  • The server list and community information cannot be loaded. Skins cannot be downloaded.
  • Connecting to most servers is not possible. Servers must be compiled with support for WebSockets.
  • No settings, demos, maps or any other files are saved when the client is closed.

You can drop a .demo file, .map file or a ddnet:// link here, or click to select a file. You can also drop files and links directly into the window after starting the client.
